# Lost & Found (Java)

A desktop GUI application for managing lost and found items on a campus, built with **Java AWT/Swing**.

## Features

- **Report** — log a lost/found item (name, category, location, date, contact)
- **Search** — find items by category, location, date, or name
- **Contact** — view claim location info and procedures

Data is persisted to a flat CSV file (`items.txt`). No external libraries required — compile and run with `javac`/`java`.
